## 📦 打包测试流程

### 第1步：构建前端

```bash
npm run build
```

**检查点：**
- [ ] 构建过程无报错
- [ ] 生成 dist/ 目录
- [ ] dist/ 目录下有 index.html
- [ ] dist/ 目录下有 assets/ 文件夹
- [ ] 资源文件正确加载

### 第2步：Electron开发模式测试

```bash
npm run electron:dev
```

**检查点：**
- [ ] Vite开发服务器启动成功（http://localhost:5173）
- [ ] Electron窗口正常打开
- [ ] 应用界面正常显示
- [ ] 功能同浏览器开发模式
- [ ] 开发者工具可正常打开（Cmd+Option+I）

### 第3步：完整打包

```bash
# 清理旧构建
npm run clean:build

# 执行打包
npm run electron:build:mac
```

**检查点：**
- [ ] 打包过程无错误
- [ ] 生成 release/ 目录
- [ ] 生成 .dmg 文件
- [ ] 生成 .zip 文件
- [ ] 文件大小合理（通常100-200MB）

### 第4步：DMG安装测试

1. 找到 release/ 文件夹中的 .dmg 文件
2. 双击打开
3. 拖拽到应用程序
4. 从启动台/应用程序启动应用

**检查点：**
- [ ] DMG打开正常
- [ ] 应用安装成功
- [ ] 应用能正常启动
- [ ] 启动时系统检测正常
- [ ] 无安全警告（或能正常打开）
- [ ] 功能全部正常
